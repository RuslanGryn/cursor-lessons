Перелік основних команд, для git системи

  • git config --global user.name : Внесення імені в розробника в проект
  • git config --global user.email : Внесення email розробника у проект
  • git init : init - Від слова initialize ( вмикати або запускати), створює новий репозиторій або оновлює вже створений
  • git status : Статус показує які дії вудбулись із репозиторієм або файлом. Якщо файл відображається червоним кольором - це доданий новий або видалений файл. Якщо колір зелений це оновлення та відслідковування файлу.
  • git add : add це збереження інформації поточного проекту та комітів
  • git commit : Це те саме що зафіксувати. Тобто зберегти зліпок проекту на певній контрольній точці та можна позначити зміти -m "Зміни".
  • git log : Одна із функці це перегляд історії змін комітів. Є можливість
  • git checkout : Ця команда дозволяє переключатися між вашими робочими вітками та отримання їх вмісту
  • git branch : З перекладу гілка. Тобто ця команда може створювати гілки, змінювати та видаляти їх.
  • git push : Використовується для встановлення зв'язку з репозиторієм та вичисленням що в ньому змінилось і відправку нових даних
  • git pull : Тобто git отримає зміни із заданого віддаленого сховища та спробує відправити їх до поточної гілки.
